WanderWise Travel App

Sofware Development Training Program - Final Project

WanderWise is an AI-assisted travel planning web app that helps users discover destinations, organize trips, and build personalized itineraries. From the home page, users can explore destination ideas without creating an account by submitting a short preference form, which the backend uses to generate five tailored travel options. Logged-in users can create trips, edit existing ones, and manage all their points of interest through editable category tables.

Each trip also includes an interactive map view, where users can visualize and manage their markers directly on the map. They can search for places, add new points of interest by clicking anywhere, and sync changes instantly with the tripâ€™s tables. When users request suggestions, they answer a few quick questions, and the backend queries the Overpass API for matching places near their existing markers; OpenAI then selects the most relevant results to send back to the UI. Users can also request AI-generated travel tips based on their current itinerary.

Tech stack:

  • Backend: Python, Flask, SQLAlchemy, SQLite, OpenAI API, Overpass API
  • Frontend: JavaScript, React, Vite, Tailwind CSS, React Leaflet + MapTiler
  • Authentication: Access token and refresh token
